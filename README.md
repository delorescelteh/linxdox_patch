# linxdox_patch

Long-term maintenance & patch project for the original Linxdot release:

- **Linxdot 2.0.0.05-OPEN A01 r17782-d6ec4dd717**

## Goals
- Maintain a curated set of patches to improve reliability in customer environments (often offline / intranet-only).
- Provide a human+AI operator workflow (docs + templates).
- Track issues and patch decisions with reproducible verification.
- Manage patch versioning and packaging.

## Repository structure
- `docs/` operator & co-work documentation
- `issues/` file-based issue records (one per incident)
- `patches/` patch specs + scripts
- `releases/` packaging & delivery notes
- `memory/` project daily notes
